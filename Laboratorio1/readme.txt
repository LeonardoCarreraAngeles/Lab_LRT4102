Introducción a Python
Python es un lenguaje de programación versátil y fácil de aprender, ideal para principiantes y profesionales. Su sintaxis es clara y legible, lo que permite escribir código de manera rápida y eficiente. Python es interpretado, lo que significa que no necesita compilación, y es multiparadigma, soportando programación estructurada, orientada a objetos y funcional.

Tipos de variables
En Python, las variables no requieren declaración de tipo. El tipo se infiere automáticamente según el valor asignado. Los tipos básicos incluyen:
- Enteros (int): Números sin decimales
- Flotantes (float): Números con decimales
- Cadenas (str): Texto entre comillas
- Booleanos (bool): Valores True o False.
- Listas (list): Colecciones ordenadas y modificables

Paradigma de Programación Orientada a Objetos (POO)
La POO es un paradigma que organiza el código en "objetos", que son instancias de "clases". Una clase es una plantilla que define atributos (datos) y métodos (funciones). La POO en Python es intuitiva y flexible. Permite modelar problemas del mundo real de manera organizada. En resumen, Python combina simplicidad y potencia, siendo ideal para aprender POO y aplicarla en proyectos pequeños o grandes. Su enfoque en la legibilidad y su amplia comunidad lo convierten en una excelente opción para programadores de todos los niveles.


Diccionarios (dict): Pares clave-valor (ej. {"nombre": "Juan"}).
Problema 1: El programa solicita un número entero positivo al usuario. Si es válido, calcula la suma de los primeros n enteros usando la fórmula matemática que se muestra en la imagen. Finalmente, muestra el resultado. Este enfoque es eficiente y evita la necesidad de bucles, aprovechando una fórmula directa para obtener la suma.
Problema 2: El programa define una lista de operadores con su nombre, sueldo por hora y horas trabajadas. Calcula la paga de cada operador multiplicando su sueldo por las horas trabajadas. Luego, solicita al usuario sus horas y costo por hora para calcular su paga. Finalmente, muestra los pagos de los operadores y la paga del usuario.
Problema 3: El programa define una lista de números. Recorre la lista para sumar los números pares y contar cuántos hay, calculando su promedio. Para los impares, calcula su producto multiplicándolos. Si no hay números pares, el promedio es 0. Finalmente, imprime el promedio de los pares y el producto de los impares.
Problema 4: El programa genera un número aleatorio entre 1 y 10. El usuario intenta adivinarlo, y el programa indica si el número es demasiado alto o bajo. El bucle continúa hasta que el usuario adivina correctamente. Al final, muestra el número de intentos. Este enfoque es interactivo y utiliza un bucle para manejar múltiples intentos.
Problema 5: El programa genera una matriz 5x5 con obstáculos aleatorios. El robot intenta llegar desde (0,0) hasta (4,4), avanzando y girando cuando encuentra obstáculos. Si no puede llegar, imprime un mensaje de error. Si lo logra, muestra el mapa original, el camino seguido y la ruta con flechas. Este enfoque simula el movimiento del robot en un entorno con obstáculos.
